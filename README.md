# m3-cc-watcher
Codelab for M3: Hands on Typescript


### Install dependencies

```sh
$ yarn install
```

### Watcher

```sh
$ yarn watch
```

### Build dist
```sh
$ yarn build
```

### Run demo
```sh
# watch Etherum to Euro price
$ node dist ETH-EUR

# watch Bitcoin to US Dollar price
$ node dist ETH-USD
```
